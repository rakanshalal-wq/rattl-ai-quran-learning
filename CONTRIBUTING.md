# Contributing

Thank you for your interest in Rattl.

This repository is currently a public showcase only. The main product source code and private MVP implementation are not included here.

## What Can Be Suggested

Public suggestions may focus on:

- Documentation clarity
- Public presentation
- Product explanation
- Showcase structure
- Typo fixes
- Safe README improvements

## What Is Not Accepted Here

Please do not submit or request changes related to:

- Backend implementation
- API routes
- Database schema
- Payment logic
- AI evaluation logic
- Environment variables
- Production infrastructure
- Real user data

## Pull Requests

Pull requests should stay public-safe and avoid exposing private product details.

Before opening a pull request, make sure your change does not include secrets, internal URLs, credentials, private screenshots, or sensitive data.
