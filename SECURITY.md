# Security Policy

This repository is a public showcase for Rattl. It does not contain the private product implementation, backend code, database schema, payment logic, AI evaluation logic, or production infrastructure.

## Reporting Sensitive Information

If you notice any accidentally exposed sensitive information in this public showcase repository, please report it privately to the repository owner instead of opening a public issue.

Sensitive information may include:

- API keys
- Environment variables
- Internal infrastructure details
- Payment identifiers
- Real user data
- Private screenshots
- Production logs

## Scope

This public repository includes documentation and presentation materials only.

The private Rattl MVP is not open for public security testing through this repository.

## Public Issues

Please do not post secrets, private data, or vulnerability details in public GitHub issues or discussions.
