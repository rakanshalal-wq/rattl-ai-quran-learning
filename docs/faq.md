# FAQ

## Is this repository the full Rattl product?

No. This is only a public showcase repository.

## Is the backend included?

No. Backend code, API routes, database schema, payment logic, AI evaluation logic, and production infrastructure are private.

## Is this an open-source product?

No. This repository is public for product presentation only. It is not a license to copy or reuse the private product implementation.

## Who is Rattl designed for?

Rattl is designed for children, parents, and Quran learning environments.

## What is the goal?

The goal is to make daily Quran memorization practice easier to follow, more structured, and clearer for parents.
