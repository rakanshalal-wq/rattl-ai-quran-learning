# Documentation

This folder contains public-safe documentation for the Rattl showcase repository.

## Files

- `product-vision.md` — public product vision
- `user-flow.md` — high-level user flow
- `roadmap.md` — public roadmap
- `faq.md` — common questions
- `public-scope.md` — what this repository includes and excludes
- `screenshots-guide.md` — safe rules for future screenshots

The private Rattl MVP implementation is not included in this repository.
