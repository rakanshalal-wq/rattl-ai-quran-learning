# Public Scope

This repository is intentionally limited to public-safe product information.

## Included

- Product overview
- Product vision
- User flow
- Roadmap
- FAQ
- Future safe screenshots or demo materials

## Not Included

- Production source code
- Backend logic
- Database schema
- Payment integration
- AI evaluation logic
- Private user data
- Environment variables
- Internal infrastructure details

## Reason

Rattl is currently a private MVP. This public repository exists to present the product direction without exposing private business or technical implementation details.
