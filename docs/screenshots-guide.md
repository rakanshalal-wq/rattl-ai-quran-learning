# Screenshots Guide

Use only public-safe screenshots in this repository.

## Safe Screenshots

- Empty parent dashboard mockups
- Child session mockups without real names
- Public landing page sections
- Pricing page without internal IDs
- Product diagrams created for showcase purposes

## Do Not Upload

- Real user names
- Real phone numbers or emails
- Payment screens with IDs
- Railway, Vercel, Supabase, or dashboard screenshots showing private details
- API keys or environment variables
- Internal logs
- Private student data

## Recommended Folder

Use a folder named:

```text
screenshots/
```

Suggested files:

```text
screenshots/parent-dashboard.png
screenshots/student-session.png
screenshots/product-overview.png
```
