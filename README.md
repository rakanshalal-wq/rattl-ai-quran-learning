# Rattl — AI-Assisted Quran Learning Platform

**Rattl** is a private MVP for an AI-assisted Quran memorization platform designed for children, parents, and Quran learning centers.

This repository is a **public showcase only**.
The main product code, backend, database, payment system, and AI evaluation logic are private.

---

## Vision

Rattl aims to make Quran memorization easier to follow, more engaging for children, and clearer for parents.

The platform is designed around a guided recitation experience where the child practices in a simple session, while parents can track progress from a dedicated dashboard.

---

## What Rattl Focuses On

* Child-friendly Quran memorization sessions
* Parent dashboard for progress tracking
* Audio-based learning experience
* Simple daily practice flow
* AI-assisted recitation feedback
* Arabic-first user experience
* Designed for families and Quran learning environments

---

## User Experience

### For Children

A clean and simple session focused on:

* Listening
* Reciting
* Repeating
* Progressing step by step

### For Parents

A dashboard to help parents:

* Follow memorization progress
* View student activity
* Manage children accounts
* Understand learning status clearly
